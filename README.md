### Manual Installation

1. Download the latest `.vsix` from the [GitHub Releases](https://github.com/HighMark-31/TRAE-Discord-Rich-Presence/releases) page.  
2. Open TRAE , go to **Extensions ‚Üí Install from VSIX...**, and select the downloaded file.  
3. Reload TRAE to activate the extension.

### Build from Source

1. Clone the repository:

```bash
git clone https://github.com/HighMark-31/TRAE-Discord-Rich-Presence.git
````

2. Navigate to the project directory:

```bash
cd TRAE-Discord-Rich-Presence
```

3. Install dependencies:

```bash
npm install
```

4. Build the project:

```bash
npm run package
```
5. Build the extension :

```bash
npx vsce package
```

6. Install the generated `.vsix` as above.
